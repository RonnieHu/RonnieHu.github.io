探索Linux系统

综合 2018/12/03

我们知道Linux是一个类Unix的操作系统，而Unix是贝尔实验室的一个研究项目，加州大学伯克利分校对其重新实现后的版本称为BSD(Mac OS部分基于BSD)。Linux是对Unix的重新实现，可以看做Unix的一个变体，但从开放形式和最终源码看，Linux不属于BSD和AT&T(贝尔实验室)风格的任何一种，因此严格来说，Linux是有别于Unix的另一种操作系统。

Linux基于GPL协议，该协议是GNU计划的一部分，此处向开源界致敬、向开源精神致敬！Linux版本众多，其中桌面常使用Ubuntu和Debian，服务器常使用Red Hat和Fedora。

常见软件包格式：.rpm(RedHat Package Manager)和.deb(for Debian & Ubuntu)；常见包管理工具APT(Advanced package Tool，可以管理rpm和deb)和yum(yum只用于rpm)。

在输入文件名/命令时，只需输入前面几个字符，然后按Tab键，shell会自动补齐。如果前面字符开头的文件/命令不止一个，可以连续按两次Tab，shell会给出所有键入字符开头的文件。

Linux启动过程：BIOS上电自检 => 引导程序装载 => 内核初始化 => 启动systemd(所有进程之父)

Bash命令：用户管理 | 软件包管理 | 进程管理 | 文件管理 | 磁盘管理 | 其他

vi模式：命令行模式/文本输入模式/末行模式

命令行模式：无论何时，按Esc即可进入命令行模式。
文本输入模式：插入命令i/附加命令a/打开命令o/修改命令r/替代命令s都可以进入文本输入模式。若想回到命令行模式，按Esc。
末行模式：在命令行模式下输入:进入末行模式。:q退出/:wq保存后退出/:q!不保存退出

Shell是解释器而不是语言，务必时刻注意Shell的代码规范。
